# 🚀 mifetch

**mifetch** — это экстремально быстрый, легкий и оптимизированный скрипт для вывода системной информации, написанный на чистом Bash.



## ✨ Особенности

*   **Минимализм:** Один из самых маленьких фетчей в мире по количеству строк и весу кода.
*   **Производительность:** Максимально оптимизирован. Мы используем встроенные возможности Bash и чтение напрямую из системных файлов `/proc`, избегая тяжелых внешних вызовов.
*   **Дизайн:** Красивый градиентный вывод в ледяных голубых тонах.
*   **Вдохновение:** Скрипт создан под вдохновением от легендарного `pfetch`, но с упором на еще большую компактность и кастомный визуальный стиль.

## 🛠 Установка

Вы можете установить **mifetch** одной командой:

```bash
curl -sSL https://raw.githubusercontent.com/RealCodeCreate/mifetch/main/install.sh | bash
```

## 🚀 Использование

После установки просто введите в терминале:

```bash
mifetch
```

## 📊 Сравнение
В отличие от многих других инструментов, **mifetch** не требует установки Python, Perl или сложных библиотек. Он работает везде, где есть Bash и `printf`.

*   **Вес кода:** < 1 КБ (982Б).
*   **Зависимости:** 0 (только базовые утилиты Linux).
