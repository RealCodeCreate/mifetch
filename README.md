# 🚀 mifetch

**mifetch** - это экстремально быстрый, легкий и оптимизированный скрипт для вывода системной информации, написанный на Rust. Также присутствует автоматическая сборка через curl.

## ✨ Особенности

* **Минимализм:** Один из самых маленьких системных фетчей по количеству строк и объему кода.
* **Производительность:** Максимально оптимизирован. Исключение запуска тяжелых внешних процессов для ускорения работы.
* **Дизайн:** Лаконичный градиентный вывод в ледяных голубых тонах.
* **Вдохновение:** Скрипт создан под вдохновением от легендарного `pfetch`, но с упором на еще большую компактность и уникальный визуальный стиль, а также перенос скрипта на Rust.

## 🛠 Установка

Вы можете установить **mifetch** одной командой (Обязательно должен быть установлен **Rust**):

```bash
curl -sSL https://raw.githubusercontent.com/RealCodeCreate/mifetch/main/install.sh | bash
```

## 🚀 Использование

После установки просто введите в терминале:

```bash
mifetch
```

## 📊 Сравнение

В отличие от многих других инструментов, **mifetch** требует только установки Rust, а также он очень быстр из за особенностей языка Rust.

* **Скорость:** На моей системе запуск был около 1.09мс, когда тот же fastfetch - 30.72мс, а pfetch - 4.76мс.
* **Зависимости:** Rust.
