# 🚀 mifetch

**mifetch** - это экстремально быстрый, легкий и оптимизированный скрипт для вывода системной информации, написанный на чистом Bash.

<img width="374" height="132" alt="image" src="https://github.com/user-attachments/assets/f2f348fd-04f3-4fab-ac39-adea891ef3e6" />

## ✨ Особенности

* **Минимализм:** Один из самых маленьких системных фетчей по количеству строк и объему кода.
* **Производительность:** Максимально оптимизирован. Используются встроенные возможности Bash и чтение напрямую из системных файлов `/proc`, что исключает запуск тяжелых внешних процессов.
* **Дизайн:** Лаконичный градиентный вывод в ледяных голубых тонах.
* **Вдохновение:** Скрипт создан под вдохновением от легендарного `pfetch`, но с упором на еще большую компактность и уникальный визуальный стиль.

## 🛠 Установка

Вы можете установить **mifetch** одной командой:

```bash
curl -sSL https://raw.githubusercontent.com/RealCodeCreate/mifetch/main/install.sh | bash
```

## 🚀 Использование

После установки просто введите в терминале:

```bash
mifetch
```

## 📊 Сравнение

В отличие от многих других инструментов, **mifetch** не требует установки Python, Perl или сложных зависимостей. Он работает везде, где есть Bash и `printf`.

* **Вес кода:** менее 1 КБ (всего 982 Б).
* **Зависимости:** отсутствуют (используются только базовые утилиты Linux).
